> **OGame Nexus** is a feature-rich tool packed into a browser extension that transforms your OGame experience into a high-tech tactical simulation. Featuring a beautiful glassmorphic dark interface, real-time scraping, complex database architectures, interactive charts, and advanced optimization algorithms.

---

## 🌟 Key Features

### 1. 📊 Empire 360 & Analytics
* **Empire Dashboard:** Get a holistic view of your entire space empire—building progress, tech queues, fleet distributions, and defense levels across all planets and moons in one beautiful interface.
* **Interactive charts:** Visual analytics built with **Recharts** representing resources, expedition outcomes, combat logs, and debris field statistics.
* **IndexedDB Storage:** Powerful client-side database management utilizing **Dexie.js** for blistering-fast local data indexing and retrieval.

### 2. 🚀 Expedition Tracking & Planning
* **Automatic Scraping & Logging:** Seamlessly records every expedition, capturing metal, crystal, deuterium, dark matter, ships gained or lost, and items found.
* **Expedition Calculator:** An advanced simulator that models class modifiers, system speeds, and lifeform bonuses to recommend the absolute optimal ship distributions for expeditions.

### 3. 🧬 Lifeform & Tech Optimizers
* **Lifeform Tech Optimizer:** Dynamically tracks population growth, building multipliers, and active lifeform skills.
* **Amortization Engine:** Recommends your next building and technology upgrades sorted strictly by payback speed. Never waste resources on low-yield upgrades again.
* **Plasma Tech ROI:** Calculates the exact resource return of upgrading Plasma Technology across all colonies, telling you exactly when and where a level-up becomes mathematically profitable.

### 4. ⚔️ Combat & Debris Logistics
* **Combat Analyzer:** Aggregates combat logs, calculates net losses/gains, counts recycled debris, and tracks overall combat profitability.
* **Debris Field Tracker:** Plots harvesting history and tracks high-yield debris field coordinates so you can harvest them with military efficiency.
* **ACS Splitter:** An Alliance Combat System split utility that accurately calculates fleet risk, fuel consumption, and profit division among multiple participants.

### 5. 🛠️ Special Utilities
* **Scrap Merchant Optimizer:** Analyzes local planetary defenses and fleets to help you trade with the Scrap Merchant for maximum return.
* **Discoverer Optimizer:** Tailors exploration profiles for the Discoverer class to maximize high-value returns.
* **Signature Forge:** A premium 3D graphic generator utilizing **React Three Fiber (R3F) & Three.js** to render breathtaking player statistics signatures that you can share in forums and Discord.

### 6. 🔒 Premium Data Sync & Privacy
* **Google Drive Sync:** Keep your local IndexedDB backed up and synchronized across multiple machines using secure Google Drive OAuth2 integration.
* **Local First:** All your data remains stored inside your browser. No external telemetry or third-party servers.

---

## 🛠️ Technology Stack

* **Core Framework:** React 18, Vite, TypeScript
* **Styling & Motion:** Vanilla CSS (Glassmorphism design tokens), Framer Motion
* **Database:** Dexie.js (IndexedDB wrapper)
* **Graphics & Charts:** Three.js, React Three Fiber (`@react-three/fiber`, `@react-three/drei`), Recharts
* **Extension Standard:** Manifest V3

## 📦 Installation & Setup

### Development Build
1. Clone the repository:
   ```bash
   git clone https://github.com/uppermosten94/OGNexusExtension.git
   cd OGNexusExtension
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build the production package:
   ```bash
   npm run build
   ```

### Loading in Browser
1. Open your browser and navigate to the Extensions page:
   * Chrome: `chrome://extensions/`
   * Edge: `edge://extensions/`
2. Enable **Developer mode** (top-right toggle).
3. Click **Load unpacked** and select the `/dist` folder generated by the build process.

---

## 🔒 Security & Privacy
OGame Nexus respects your privacy. All empire statistics, coordinates, and data are stored locally in your browser's IndexedDB. Cloud synchronization is entirely opt-in and operates directly between your browser and your personal **Google Drive** storage space.
